# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Using as Tools:

- [] Vite + TypeScript
- [] Styled components
- [] Radix UI
- [] Json-server
- [] React-router-dom
- [] Phosphor icons
- [] React-hook-form
- [] Zod
- [] Axios

## Expanding the ESLint configuration

Start with:

```
npm i
npm run dev
npx json-server server.json -p 3333
```
